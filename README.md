# 📊 Dividend Tracker App

A sleek tool for tracking monthly dividend payouts, visualizing ETF performance, and simulating reinvestment strategies—built to help users make smarter investing decisions.

## 🚀 Features

- ETF snapshot view with price & performance indicators
